# Arithmetic Operators


# Loops

```py
if __name__ == '__main__':
    n = int(input())
    
    for i in range(n):
        print(i**2)
```

# Print
  
```py
if __name__ == '__main__':
    n = int(input())
    
    ans = ''
    
    for i in range(1,n+1):
        ans += str(i)
    
    print(ans)
```
